# Counter App – Machine Coding

## Overview
This is a simple **Counter Application** built as part of **machine coding interview practice**.  
It focuses on clean implementation, minimal UI, and clear state management.

This problem is commonly asked in frontend machine coding rounds to evaluate:
- Understanding of component state
- Logical thinking and UI updates
- Code clarity under time constraints

---

## Features
- Increment counter value
- Decrement counter value
- Reset counter to initial state
- Minimal and clean user interface
