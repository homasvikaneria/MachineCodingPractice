# User Search App – Machine Coding

## Overview
This is a simple **User Search Application** built as part of **machine coding interview practice**.  
It focuses on fetching data from an API and implementing real-time search filtering.

This problem is commonly asked in frontend machine coding rounds to evaluate:
- API data fetching
- State management
- Search and filter logic
- Code clarity under time constraints

---

## Features
- Fetch user data from external API
- Search users by name
- Real-time filtering based on input
- Clean and minimal user interface
