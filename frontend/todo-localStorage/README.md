# Todo List App – Machine Coding

## Overview
This is a simple **Todo List Application** built as part of **machine coding interview practice**.  
It focuses on clean implementation, minimal UI, and basic state persistence.

This problem is commonly asked in frontend machine coding rounds to evaluate:
- Understanding of component state
- Logical thinking and UI updates
- Code clarity under time constraints

---

## Features
- Add new todo item
- Mark todo as complete / incomplete
- Delete todo item
- Persist data using localStorage
- Minimal and clean user interface
